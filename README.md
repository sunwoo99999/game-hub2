# game-hub2

## Introduction

This project is a web-based video game discovery application developed using TypeScript, Vite, and React. It is designed to help users browse and explore a wide variety of video games with an efficient and responsive user interface.

## Features

  * **Game Discovery**: Browse a grid of video games with essential details.
  * **Filtering System**: Filter games by genre and platform to find specific titles.
  * **Sorting Options**: Sort games based on criteria such as relevance, release date, or popularity.
  * **Responsive Layout**: Fully optimized design that works seamlessly on desktop and mobile devices.
  * **Dark Mode Support**: Toggle between light and dark themes for visual comfort.
  * **Modern Tech Stack**: Built with TypeScript and Vite for fast performance and type safety.

## Installation

Follow the steps below to set up and run the project locally.

1.  **Clone the repository**
    git clone [https://github.com/sunwoo99999/game-hub2.git](https://www.google.com/search?q=https://github.com/sunwoo99999/game-hub2.git)

2.  **Navigate to the project directory**
    cd game-hub2

3.  **Install dependencies**
    npm install

4.  **Start the development server**
    npm run dev
